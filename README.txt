📊 Data Structures & Algorithms in Python

A curated collection of core data structures and algorithms implemented
from scratch in Python 🐍

This repository is designed to build a deep, practical understanding of
algorithms through clean implementations and hands-on learning.

It focuses on essential problem-solving techniques such as: - 🔁
Recursion - ✂️ Divide & Conquer - 🧠 Dynamic Programming - 🌐 Graph
Algorithms

------------------------------------------------------------------------

🎓 COURSE ORIGIN

This project is based on the Udemy course:

“The Complete Data Structures and Algorithms Course in Python” by Elshad
Karimov

All implementations follow the course structure while emphasizing: - ✨
Clean and readable code - 🧩 Modular design - 🛠️ From-scratch
implementations (no external libraries)
